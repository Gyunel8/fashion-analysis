# Fashion Sales & Customer Analytics

This is an interactive dashboard I built to analyze retail data. My main goal was to turn raw transactional data into clear, actionable insights for decision-making.

## What I did:
* **Data Prep:** Cleaned and pre-processed the retail dataset using Python (Pandas). I focused on handling missing values and ensuring the data was ready for analysis.
* **Analysis:** Analyzed revenue trends across different brands and examined the main reasons for product returns.
* **Visualization:** Built an interactive dashboard using Streamlit to make the data easy to explore for stakeholders.

## Technical Skills Used:
* Python (Pandas, Plotly)
* Streamlit for building the web interface
* Data cleaning and exploratory data analysis (EDA)
